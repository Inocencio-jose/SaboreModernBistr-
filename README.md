# Saboré Modern Bistro - Menu Digital

Site pronto para deploy no Netlify. Single page app (uma página) com tabs e filtro JS.

## Como usar
1. Baixe este diretório inteiro (ou copie os arquivos para uma pasta local).
2. No Netlify: arraste a **pasta** `menu-sabore-modern-bistro` para a área de deploy (ou conecte via Git).
3. Opcional: substitua as imagens em `assets/img/` pelas imagens reais do restaurante.
4. Gere um QR Code apontando para a URL publicada no Netlify.

## Notas técnicas
- Tailwind via CDN (rápido e sem build). Se preferir um build com Tailwind (mais customizável), posso fornecer instruções.
- JavaScript puro: troca de tabs, filtro por pesquisa, animações leves e lazy-loading de imagens.
